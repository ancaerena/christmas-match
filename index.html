Welcome to the Christmas Memory Game


This matching game will put your memory to test. There are 9 images hiden behind these cards.
Each image has its matching pair hidden on the board. You can turn only two cards at once, if they match, they will stay turned.
If not, you can try again, finding each card's match.
Let the fun begin by clicking on the first card you want to turn!
